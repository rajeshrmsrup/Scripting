Formatting Tags

<b> tag defines bold text, without any extra importance

<strong> tag defines text with strong importance. The content inside is typically displayed in bold

<i> tag defines a part of text in an alternate voice or mood. The content inside is typically displayed in italic

<em> tag defines emphasized text. The content inside is typically displayed in italic

<small> tag defines smaller text

<mark> tag defines text that should be marked or highlighted

<del> tag defines text that has been deleted from a document

<strike> tag is used to draw a strikethrough on a section of text

<ins> tag defines a text that has been inserted into a document

<sub> tag defines subscript text: like H2O

<sup> tag defines superscript text: like (a+b)2=a2+b2+2ab

<u> tag is used to underline text

<big> tag is used to increase the font size by one conventional unit
